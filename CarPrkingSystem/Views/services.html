Services

Owner

1. Space Owner can register in the website.
2. He/She will carry an unique id number given by the website.
3. Space Owner can update his spaces according to his/her needs.
4. Space Owner can report to the admin if any Parker missbehave or miss-use the system.

Parker

1. Parker can register in the website.
2. He/She will carry an unique id number given by the website.
3. Parker can input for how many hours he/she will hold the space and for that he/she will be charged which will be pre-defined in the wesite as per minuite.
4. Parker can discharge the place whenever he/she wants.
5. Parker can report to the admin if any Space Owner missbehave or miss-use the system.
